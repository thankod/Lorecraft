# 第二层：浮现特质系统

## 性质

- 非预设，由玩家行为模式驱动，动态浮现与消退
- 有鲜明个性，声音风格各异
- 是玩家行为模式的镜像——玩家怎么玩，角色内心就住进什么

---

## 特质分类

特质分为两大类，对应不同的驱动信号：

### 表达类特质（由信号 A 驱动：输入语气/风格）

玩家持续以某种语气输入，对应特质逐渐浮现。

| 特质 | 触发模式 | 声音风格 |
|------|----------|----------|
| 戏谑 | 频繁使用讽刺、轻浮、调侃语气 | 油嘴滑舌，自我娱乐 |
| Horny | 频繁对特定对象表达性/浪漫意图 | 直白、不加掩饰 |
| 批评 | 频繁对 NPC 或处境表达否定评价 | 刻薄、精准 |
| 傲慢 | 频繁以俯视姿态表达意图 | 不耐烦，居高临下 |

### 价值观类特质（由信号 B 驱动：关键节点的实际选择）

玩家在重要决策点反复做出同类选择，对应特质逐渐浮现。

| 特质 | 触发模式 | 声音风格 |
|------|----------|----------|
| 无情 | 反复在利益冲突中牺牲他人 | 冷酷、效率至上 |
| 冲动 | 反复在警告后仍强行推进 | 兴奋、短视 |
| 偏执 | 反复回到同一个目标/执念 | 低沉、固执 |
| 共情 | 反复主动关注 NPC 的处境 | 柔软、敏感 |

---

## 浮现机制

特质的浮现是**悄无声息**的——没有提示，没有仪式感。

```
权重低于下限阈值  → 特质沉默（不存在）
权重超过浮现阈值  → 特质开始发言（第一次出现，无任何说明）
权重持续增长      → 发言频率和强度增加
权重下降至沉默阈值 → 特质逐渐停止发言，直至消退
```

第一次浮现时，声音直接说话，不自我介绍。
玩家若注意到，是因为他们感受到了，而不是被告知。

---

## 特质消退

特质消退同样是渐进的：
- 不做触发行为 → 权重自然衰减（指数衰减）
- 做出相反行为 → 权重主动抵消
- 消退过程中发言频率降低，最终沉默

特质消退不等于"被治愈"——如果玩家重新走回老路，权重会重新积累，且积累速度可能更快（路径依赖效应，待定）。
