# 第一层：底层认知功能

## 性质

- 恒常存在，不会消退
- 功能性、相对冷静，是角色理性/直觉的基础载体
- 声音风格稳定，但可随角色成长发生细微变化

---

## 预设认知功能（示例，可扩展）

| 名称 | 职责 | 典型发言风格 |
|------|------|-------------|
| 反思 | 回顾过去的选择与后果，识别模式 | 冷静、略带疲惫 |
| 规划 | 评估当前意图的可行路径与代价 | 分析性、直接 |
| 探究 | 对未知信息产生好奇，推动信息收集 | 好奇、追问 |
| 直觉 | 提供无法言明的感受性判断 | 模糊、意象性 |
| 记忆 | 在相关情境下唤起过去经历 | 片段化、感性 |

---

## 与第二层的关系

- 第一层不会被第二层压制，但会被第二层打断或影响优先级
- 辩论中，第一层往往扮演"理性锚点"角色
- 第一层的存在保证了角色不会因第二层特质的极端化而失去基本认知能力
